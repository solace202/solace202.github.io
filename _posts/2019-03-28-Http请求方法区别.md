### 概念

* 副作用

  当发送完一个请求后，网站上的资源状态没有发生修改，即认为这个请求是无副作用的。

* 幂等性

  一个请求无论发送多少次，服务器上资源的状态最终是一致的。

* request body

  请求可传输的数据量

### http method 区别

| method | 副作用 | 幂等性 | request body |
| ------ | ------ | ------ | ------------ |
| GET    | 无     | 幂等   | 不可以带     |
| POST   | 有     | 非幂等 | 可以带       |
| PUT    | 有     | 幂等   | 可以带       |
| DELETE | 有     | 幂等   | 不可以带     |
